# 🎨 van.ali.7 - Générateur d'Images IA

![van.ali.7](https://via.placeholder.com/800x400/667eea/ffffff?text=van.ali.7+AI+Generator)

**van.ali.7** est une application web moderne et élégante qui génère des images par intelligence artificielle. Conçue avec une interface sombre (Dark Mode) et optimisée pour les appareils Android.

## ✨ Fonctionnalités

- 🧠 **Modèles IA Multiples** : Choisissez entre Flux, Turbo, Realism, Anime, 3D, et plus.
-  **100% Mobile** : Interface responsive, parfaitement adaptée aux smartphones Android.
- 🌙 **Design Moderne** : Thème sombre avec dégradés violet/bleu et effets de verre (Glassmorphism).
- ️ **Téléchargement Rapide** : Sauvegardez vos créations en un seul clic.
- ⚡ **Gratuit & Rapide** : Propulsé par l'API publique Pollinations AI.
